What is CSS?

CSS stands for Cascading Style Sheets. It is used to style the look of your webpage.

h1 {
  color: blue;
  font-size: 36px;
}

p {
  font-family: Arial;
  color: gray;
}

This CSS code styles headings and paragraphs with color and font settings.
